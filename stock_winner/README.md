# StockWinner-backend

## Development

### Backend
```$ gradle build```

```$ java -jar build/libs/stockwinner-0.0.1.jar```

Server listen at localhost:8080

### Frontend

```$ npm install```

```$ npm start``` 

Server listen at localhost:4200

### Database 

```$ sudo ./docker_init.sh```

## Build Fat Jar
```$ gradle clean build```

```$ java -jar stockwinner-0.0.1.jar```
